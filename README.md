# Автоматизация обработки отчётов продаж

**Python-скрипт** для автоматического объединения нескольких Excel-файлов с продажами, расчёта маржи/прибыли и генерации красивого аналитического отчёта.

### Возможности проекта
- Объединение любого количества Excel-файлов из папки
- Автоматическая очистка данных (дубли, пропуски, некорректные значения)
- Расчёт ключевых метрик: сумма продажи, маржа, прибыль, % маржи
- Сводные таблицы по менеджерам и категориям товаров
- Топ-10 товаров по выручке
- Красивое форматирование Excel (цветная шапка, автоширина, денежный формат)
- Построение графика продаж по категориям

### Как запустить

```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Запуск
python main.py

# Запуск с параметрами
python main.py --folder reports --output мой_отчет.xlsx
